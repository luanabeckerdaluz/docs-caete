Data Sources

  • Source: ISIMIP repository (ISIMIP3a and ISIMIP3b protocols)

  • Format: NetCDF files and legacy .pbz2 files

  • Processing: Custom preprocessing scripts included

Key Components

  • Bias-corrected data from ISIMIP repo in _raw folders

  • Gridlist tables for execution control

  • Input folders: spinclim, obsclim, historical, scenarios

Data Processing Pipeline

This repository contains a folder named input which includes:

  • preprocess_caete.py: Reads ISIMIP netCDF files and creates netCDF input files for CAETÊ.

  • preprocess_caete_pbz2.py: Converts ISIMIP netCDF files into legacy .pbz2 input files.

  • The new input handler supports both formats, configurable via caete.toml.

Gridlist Generation: - Generated by input_handler class - Required for gridcell execution - Saved alongside input files - grd folder has some examples
